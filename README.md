# Lavadero — Car-Wash Cereté

Aplicación web (React + Vite + TypeScript + Tailwind) con Supabase.

El código de la app está en **`carwash-web/`**.

## Desarrollo local

```bash
cd carwash-web
npm install
npm run dev
```

Copia `.env.example` a `.env` y configura `VITE_SUPABASE_URL` y `VITE_SUPABASE_ANON_KEY`.

## Supabase

Scripts SQL y plantillas de correo: `carwash-web/supabase/`.
